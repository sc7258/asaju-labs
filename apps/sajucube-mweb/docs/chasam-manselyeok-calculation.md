# 차샘 만세력 계산 방법

## 개요

차샘 만세력은 하나의 생년월일시(양력)를 기준으로 6개의 사주 패널을 산출하는 방식이다.  
각 패널은 **차역(借曆)** 과 **부허자(附虛字)** 라는 두 가지 날짜 변환 방식을 조합해 도출된다.

---

## 6개 패널 구성

| 패널 키 | 이름 | 도출 방법 |
|---|---|---|
| `buheoja-bonwon` | 부허자 본원 | 부허자차역 결과를 다시 부허자 변환 |
| `buheoja-charyeok` | 부허자 차역 | 입력 양력 → 음력 변환 후 그 수치를 양력으로 재해석 |
| `bonwon` | 본원 | 입력받은 원래 생년월일 (변환 없음) |
| `charyeok` | 차역 | 입력 양력 수치를 음력 날짜로 해석 후 다시 양력으로 변환 |
| `heoja-bonwon` | 허자 본원 | 차역 결과를 다시 차역 변환 |
| `heoja-charyeok` | 허자 차역 | 허자 본원 결과를 다시 차역 변환 |

---

## 두 가지 핵심 변환

### 1. 차역 변환 (借曆, buildCharyeokStage)

> "날짜 숫자를 달력 체계만 바꿔서 읽는다"

1. 입력 양력 날짜의 **(년, 월, 일) 숫자** 를 **음력 날짜** 로 그대로 해석한다.  
   예: 양력 1972-01-26 → 음력 1972년 1월 26일로 읽음
2. 그 음력 날짜가 **실제로 존재** 하면 한국 음력 → 양력 변환을 그대로 적용한다.
3. **존재하지 않는 음력 날짜** (예: 음력 1월 31일) 라면 아래 규칙으로 가장 가까운 날짜를 선택한다.

#### 존재하지 않는 음력 날짜 처리 (resolveClosestChasamLunarDate)

1. 해당 음력 연도 전체의 날짜 목록을 구성하고, 목표 음력 날짜보다 작은 마지막 날짜(**이전 후보**)와 큰 첫 번째 날짜(**다음 후보**)를 찾는다.
2. 두 후보 모두 있으면 기준 날짜(입력 시각)와의 거리가 짧은 쪽을 선택한다.
3. **거리가 같으면** 생시(hour)로 판단한다:
   - `hour < 12` → 이전 후보 선택
   - `hour >= 12` → 다음 후보 선택

---

### 2. 부허자 변환 (附虛字, buildBuheojaStage)

> "음력으로 읽은 날짜를 달력 체계만 다시 바꿔서 양력으로 읽는다"

1. 입력 양력 날짜를 **한국 음력으로 변환** 한다.
2. 변환된 음력의 **(년, 월, 일) 숫자** 를 **양력 날짜** 로 그대로 해석한다.  
   예: 양력 1972-01-26 → 음력 1971-12-11 → 양력 1971-12-11로 읽음
3. 해당 양력 날짜가 **유효하지 않으면** (예: 2월 29일인데 평년) 아래 규칙으로 보정한다.

#### 유효하지 않은 양력 날짜 처리 (resolveClosestBuheojaSolarDate)

1. 날짜 오버플로우로 실제 날짜를 계산한 뒤 ±2일 범위의 후보를 생성한다.
2. 후보 중에서 **원래 목표 음력 날짜와 일치** 하는 날짜가 있으면 그것을 선택한다.
3. 일치하는 후보가 없으면 생시로 판단한다:
   - `hour < 12` → 오버플로우 날짜 기준 하루 이전 선택
   - `hour >= 12` → 오버플로우 날짜 그대로(또는 다음 날) 선택

---

## 전체 체인 도출 순서 (buildChasamDateChain)

```
입력: 양력 생년월일시 (year, month, day, hour, minute)

buheojaCharyeok = 부허자변환(입력)
buheojaBonwon   = 부허자변환(buheojaCharyeok.최종날짜)

bonwon = 입력날짜 그대로

charyeok     = 차역변환(입력)
heojaBonwon  = 차역변환(charyeok.최종날짜)
heojaCharyeok = 차역변환(heojaBonwon.최종날짜)
```

시각(hour, minute)은 체인을 따라 내려갈 때 **항상 원본 입력값** 을 유지한다.

---

## 구체적인 계산 예시

**입력**: 양력 1972-01-26 11:30

| 단계 | 처리 내용 | 결과 날짜 |
|---|---|---|
| **본원** | 변환 없음 | 양력 1972-01-26 |
| **차역** | 양력 수치 1972-01-26을 음력으로 해석 → 음력 1972년 1월 26일이 존재함 → 양력 변환 | 양력 1972-03-11 |
| **허자 본원** | 차역(1972-03-11) 수치를 음력으로 해석 → 음력 1972년 3월 11일이 존재함 → 양력 변환 | 양력 1972-04-24 |
| **허자 차역** | 허자 본원(1972-04-24) 수치를 음력으로 해석 → 양력 변환 | (연속 진행) |
| **부허자 차역** | 양력 1972-01-26을 음력으로 변환 → 음력 1971-12-11 → 양력 1971-12-11로 해석 | 양력 1971-12-11 |
| **부허자 본원** | 부허자차역(1971-12-11)을 음력으로 변환 → 음력 1971-10-24 → 양력 1971-10-24로 해석 | 양력 1971-10-24 |
