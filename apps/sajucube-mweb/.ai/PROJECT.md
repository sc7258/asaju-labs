# SajuCube Webapp

## 목표
- Next.js 기반으로 사주/만세력 기능을 제공하는 웹앱을 만든다.
- 첫 마일스톤은 출생 정보를 입력하면 만세력 표를 보여주는 기능이다.

## 현재 1차 범위
- 성별, 양력/음력, 윤달 여부, 생년월일시 입력
- 만세력 4주 표시
- 천을귀인, 공망 표시
- 대운 10칸, 세운 10칸 표시
- 모바일에서도 이질감 없는 compact UI

## 기술 스택
- Next.js App Router
- TypeScript
- Tailwind CSS
- Vitest
- Supabase
- `@gracefullight/saju`
- Luxon

## 작업 방식
- 기능 구현과 테스트를 같이 진행한다.
- UI는 참고 이미지와의 유사도를 중요하게 본다.
- 문서는 한국어 기준으로 유지한다.
