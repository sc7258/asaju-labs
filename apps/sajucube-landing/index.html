iPhone / iPad 설치 방법

  1. Safari 브라우저로 사주큐브에 접속합니다.
    (Chrome, 네이버 앱 등에서는 불가)
  2. 화면 하단 중앙의 ⍐ 공유(Share) 버튼을 탭합니다.
  3. 메뉴를 위로 스크롤하여 '홈 화면에 추가'를 선택합니다.
  4. 우측 상단의 '추가'를 누르면 바탕화면에 전용 앱이 설치됩니다!

Android (갤럭시 등) 설치 방법

  1. Chrome 브라우저로 사주큐브에 접속합니다.
  2. 화면 우측 상단의 ⋮ 메뉴 버튼을 탭합니다.
  3. 메뉴 목록에서 '홈 화면에 추가' 또는 '앱 설치'를 선택합니다.
  4. '설치' 팝업이 뜨면 확인을 눌러 완료합니다! 바탕화면에서 찾아보세요.

PC / Mac 설치 방법

  1. PC에서 Chrome 또는 Edge 브라우저로 접속합니다.
  2. 주소창(URL) 우측 끝에 나타나는 💻 설치 아이콘(앱 다운로드)을 클릭합니다.
  3. 설치 확인 팝업창에서 '설치'를 클릭합니다.
  4. 바탕화면과 시작 메뉴에 사주큐브 단축 아이콘이 생성되어 브라우저 탭이 아닌 독립된 창으로 실행됩니다!

주요 특징

⚡ Headless 엔진

화면에 종속되지 않은 순수 명리 계산 코어로, 극강의 정확도와 빠른 속도를 자랑합니다.

🎨 모던 UI/UX

한자 위주의 딱딱한 디자인을 벗어나, 모바일과 PC 모두에서 시각적으로 아름다운 인터페이스를 제공합니다.

🔗 확장성 (예정)

추후 사주덱스(Sajudex)와 연동되어 지인 그룹핑 및 유명인 사주 검색 기능까지 확장됩니다.

개발 로드맵

Phase 1 (진행 중)

고정밀 Core 명리 계산 엔진 구축 및 오행/십성 테마 UI 디자인 적용

Phase 2 (예정)

대운/세운 분석 및 신살(도화, 역마 등), 합충파해 계산 시각화 기능 오픈

Phase 3 (예정)

'사주덱스(Sajudex)' 연동 - 유명인 사주 검색 및 개인 지인 그룹핑/메모 기능 추가
