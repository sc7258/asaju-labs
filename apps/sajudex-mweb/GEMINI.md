# Sajudex MWeb - Agent Instructions

이 파일은 `sajudex-mweb` 프로젝트에서 작업하는 AI 에이전트를 위한 로컬 컨텍스트 및 필수 규칙입니다.

## 1. 프로젝트 목적
이 프로젝트는 사주덱스 생태계 중 지인의 명식을 관리하는 **인연록(Personal CRM)**의 모바일 웹 프론트엔드입니다. 

## 2. 아키텍처 제약 사항 (중요)
- **Local-First (Phase 1):** 현재 단계에서는 백엔드 서버나 DB 연동을 시도하지 마세요. 모든 데이터 읽기/쓰기는 브라우저의 `IndexedDB`를 사용합니다.
- **연락처 연동 보류:** 구글 주소록 연동이나 전용 연락처 컬럼(전화번호, 이메일)을 스키마에 추가하지 마세요. 사용자가 필요시 **'메모(TEXT/JSON)' 필드**에 자율적으로 기재하도록 폼과 UI를 구성합니다.
- **의존성 분리:** 사주 명식 계산과 관련된 핵심 로직(천간/지지 도출 등)은 가급적 이 앱 내부에 하드코딩하지 않습니다. 향후 모노레포의 `packages/saju-core`에서 가져다 쓸 수 있도록 인터페이스를 분리하거나 Mock 데이터로 먼저 작업합니다.

## 3. 스타일 및 UI/UX 규칙
- **Mobile First:** 데스크톱 브라우저에서도 최대 너비(`max-w-md`)의 컨테이너 안에서 모바일 앱처럼 보이도록 렌더링해야 합니다.
- **스타일링:** Tailwind CSS v4를 사용합니다. 별도의 무거운 UI 라이브러리(MUI, AntD 등) 도입은 지양합니다.
- 복잡한 폼(Form) 위주의 화면보다는, 빈 공간이 많고 기록하기 편한 노트(Note) 형식의 여유로운 레이아웃을 지향합니다.
