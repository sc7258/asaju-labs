# Sajudex MWeb (인연록) 개발 로드맵

이 로드맵은 Phase 1(로컬 저장)과 Phase 2(클라우드 동기화)의 구체적인 프론트엔드 개발 단계를 정의합니다.

## Phase 1: Local-First (현재 진행 중)
가입 없이 브라우저의 `IndexedDB`를 활용하여 동작하는 완전한 스탠드얼론 모바일 웹 구현.

- [x] 모바일 웹 레이아웃 및 뼈대 세팅 (Next.js, BottomNav)
- [x] IndexedDB 스토리지 레이어 구성 (`localforage` 또는 `dexie` 도입)
- [x] 인연 추가 UI (이름, 생년월일, 성별 등 기본 폼)
- [x] 인연 리스트 및 상세 페이지 UI
- [ ] 사주(만세력) 코어 패키지 연동 및 명식 렌더링 (사주큐브 로직 활용)
- [ ] 관계 메모 작성 및 JSON 데이터 백업/복원 기능 구현

## Phase 2: Cloud Sync (향후 계획)
- [ ] 카카오/구글 소셜 로그인 연동
- [ ] 서버 API 연동 (Prisma ORM 기반 메인 DB)
- [ ] 최초 로그인 시 IndexedDB 로컬 데이터를 서버로 마이그레이션(일괄 업로드) 하는 로직 개발
- [ ] 연락처 앱 등 외부 API 연동 여부 검토 (사용자 피드백 기반)
