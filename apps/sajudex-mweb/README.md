# Sajudex MWeb (인연록)

사주 기반 지인 관계 관리 모바일 웹 프론트엔드 (Phase 1: Local-First)

## 기술 스택
- Framework: Next.js (App Router)
- Language: TypeScript
- Styling: Tailwind CSS v4
- Storage: IndexedDB (Local-First)

## 실행 방법
루트 폴더(`asaju-labs`)에서 다음 명령어를 실행합니다:
```bash
pnpm turbo run dev --filter sajudex-mweb
```
